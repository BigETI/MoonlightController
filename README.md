# Moonlight controller

## Description
Moonlight controller is a cross platform library to control user I/O interfaces using a scripting backend.

## Supported languages
- C++
- Lua

## Supported platforms
- Linux *
- OSX *
- Windows

\* will be supported at a later point in time
